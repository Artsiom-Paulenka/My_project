1) Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

   Для того чтобы отказаться надо ввести команду git checkout + имя файла(файлов), где нужно откатить изменения.

2) Как вернуть файл из индекса в рабочий каталог?

  Командой git restore --staged + имя файла(файлов)

3) Как отменить ошибочный commit ?

  git reset HEAD ~ 1  откатывает на 1 коммит
