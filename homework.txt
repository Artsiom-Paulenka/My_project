1) Как скопировать на свой компьютер репозиторий из github.com, 
что должно быть сделанно предварительно?
  - создать репозиторий на GitHub
  - скопировать ссылку на этот репозиторий
  - в GitBush зайти в нужную директорию куда будет клонироваться репозиторий с GitHub 
  - ввести команду git clone и ссылку на GitHub репозиторий (git clone git@github.com:Artsiom-Paulenka/My_project.git )
  
2) Как создать git репозиторий из локального каталога на своем комьютере. Как его отправить на github.com?
  - зайти в нужную директорию на GitBush
  - с помощью команды git init + имя локального каталога инициализировать  
  - с помощью командs git add . инициализировать файлы внутри каталога
  - с помощью команды git commit -m "Add commit" добавить комментарий
  - создать в ручную репозиторий на GitHub
  - создать ветку  git branch -M main
  - связать лок.каталог  и репозиторий GitHub с помощью команды git remote add origin git@github.com:Artsiom-Paulenka/My_project.git
  - запушить ветку git push -u origin main

3) Как получить на локальный компьютер изменения с github.com
  - сделать изменения на github репозитории
  - войти в GitBush в свой локальный репозиторий
  - с помощью команды git pull синхронизировать изменения
